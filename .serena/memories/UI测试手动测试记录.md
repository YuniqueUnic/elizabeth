## 手动测试发现的 UI 元素和功能情况

### 已验证正常工作的功能：

1. **消息发送** - 可以成功发送消息
2. **消息计数** - 消息列表显示正确的消息计数（从"共 0 条"变为"共 1 条"）
3. **未保存标签** - 发送消息后显示"未保存"标签（ref=e231）
4. **保存功能** - 点击保存按钮成功保存消息，显示成功通知
5. **消息状态变更** - 保存后"未保存"标签消失

### UI 元素位置（ref 编号）：

- 顶部栏：banner (ref=e3)
- 左侧边栏：complementary (ref=e15)
- 中间主区域：main (ref=e61)
- 右侧文件管理：complementary (ref=e185)
- 消息输入框：textbox (ref=e179)
- 发送按钮：button "发送" (ref=e208 或变动的 ref)
- 保存按钮：button "保存" (变从 disabled 到 enabled)
- 消息列表：容器 (ref=e64)
